Q1 - Identify and explain common and important components and concepts of web development markup languages.

The components of markup languages consists of tags, keywords and names.
There are many uses of a tag in web developement and markup language. The important component is the document tag component. Which helps a web browser determine between simple or HTML content. Which dictates how the content is viewed, displayed, or formated on the webpage.
Some common tags include keyword tags. That are used to define, group and give more information about the webpage content.

The name component defines and contains a certain section of content on a webpage. And can be used in reference to the styling and formatting of the section.

They are usually used in specifying the different sections and grouping the specific content, under its own name within the webpage.

With these components, they allow content and information to be formatted and displayed in the style or position you want it to be. Or change how the content is viewed.
